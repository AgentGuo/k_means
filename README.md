# 基于IRIS数据集k-means算法实践

使用的是Iris数据集（鸢尾花卉数据集），是一类多重变量分析的数据集。数据集包含150个数据样本，分为3类，每类50个数据，每个数据包含4个属性，分别是花萼长度，花萼宽度，花瓣长度，花瓣宽度。使用k-means聚类算法对数据进行聚类。

使用python库的版本：

* `matplotlib`:`2.2.3`
* `numpy	`:`1.15.4`
* `sklearn`:`0.19.2`
